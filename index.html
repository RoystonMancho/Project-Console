Visual Studio

Microsoft Visual Studio is an integrated development environment (IDE) from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps. Visual Studio uses Microsoft software development platforms such as Windows API, Windows Forms, Windows Presentation Foundation, Windows Store and Microsoft Silverlight.

How to setup Visual studio

Visual studio is set up by installing it then create a new folder and download the live-server Extension.

Console

Console.log is a function that is used to print the values into the Console Screen. It is used for testing purposes. To open console you need to right click then click inspect then you click on console and you will be able to see what you printed on the Console Screen.
